DESIGN NOTES:

Modal Styling:
- Background overlay: rgba(0,0,0,0.8) - 80% black
- Modal box: #2a2a2a background, #8b0000 border
- Max width: 700px
- Max height: 80vh (scrollable if content exceeds)
- Centered on screen
- Drop shadow for depth

Item Cards:
- Background: #1a1a1a
- Border: #8b0000 (1px solid)
- Padding: 20px
- Margin between cards: 15px
- Icon size: 32px emoji or SVG
- Text: 16px serif for description

Typography:
- Title: 20px, bold, sans-serif
- Item name: 18px, bold
- Description: 16px, serif, italic
- Metadata: 14px, sans-serif, #999

Accessibility:
- Focus trap: Tab cycles within modal only
- Esc key closes modal
- Screen reader: "Inventory modal. You have 3 items. Magic Spectacles: [description]"
- Focus first item on open

- Modal 700px wide
- 3-4 items visible without scrolling

EDGE CASES:

1. Scrolling with many items:
   - Fixed header (title + close button)
   - Scrollable content area
   - Fixed footer (close button)

2. Very long item description:
   - Max height for card
   - "Read more" expansion

3. Special items (cursed, temporary):
   - Red tint for dangerous items
   - Timer icon for temporary items

4. Item consumed/lost:
   - Grayed out with "Used" label
   - Removed from inventory count
