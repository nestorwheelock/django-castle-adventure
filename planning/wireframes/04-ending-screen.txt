DESIGN NOTES:

Colors:
- Ending title: #8b0000 (dark red) or #gold for victory endings
- Achievement box: #2a2a2a background, #00ff00 (green) text
- Stats: #999 (gray) labels, #e0e0e0 (white) values
- Secret ending: Gold gradient (#FFD700)

Typography:
- Ending title: 32px, bold, centered
- Ending text: 18px, serif, justified
- Achievement: 20px, bold, centered
- Stats: 16px, monospace

Layout:
- Centered column, max-width 800px
- Generous spacing between sections
- Ending text is scrollable if very long

Animation (optional):
- Achievement box slides in from top
- Confetti effect for secret ending
- Stats count up from 0

Accessibility:
- Screen reader announces ending type first
- Full text is readable
- All interactive elements keyboard accessible

EDGE CASES:

1. Very fast playthrough (<5 min):
   - Special badge: "Speedrunner!"
   - Show time compared to average

2. Many deaths (>10):
   - Humor badge: "Persistent Corpse"
   - "You died 15 times but never gave up!"

3. Perfect run (0 deaths):
   - Special badge: "Untouchable"
   - Bonus achievement

4. Unusual stats (e.g., only 3 scenes visited but reached ending):
   - Possible if player found secret passage
   - Badge: "Shortcut Master"
