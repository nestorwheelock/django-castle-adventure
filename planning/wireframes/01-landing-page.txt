───────────────────────────────────────────────────────────────────────────────────

INTERACTIONS:

[CONTINUE ADVENTURE] Button:
- Loads saved game state
- Restores scene, inventory, flags
- Takes player to game board at current scene
- PRIMARY ACTION (highlighted in dark red)

[START NEW ADVENTURE] Button:
- If save exists: Confirms "Overwrite saved game?"
- Starts fresh at Scene 01 (Outside Castle Walls)
- Clears all progress
- SECONDARY ACTION (outlined)

[View Endings Collection] Link:
- Shows grid of all 5 endings (locked/unlocked)
- Allows player to see progress
- NON-DESTRUCTIVE (just informational)

[How to Play] Link:
- Tutorial/instructions modal
- Explains choices, inventory, save system
- Can be dismissed

───────────────────────────────────────────────────────────────────────────────────

STATES:

STATE 1: No Saved Game
- "SAVED GAME FOUND" panel is hidden
- Only [START NEW ADVENTURE] button visible
- Text changes to "BEGIN YOUR ADVENTURE"

STATE 2: Saved Game Exists (shown above)
- Both buttons visible
- Saved game details displayed
- [CONTINUE] is primary action

STATE 3: Mobile View
- Buttons stack vertically
- Smaller ASCII art
- Larger touch targets (48px minimum)

───────────────────────────────────────────────────────────────────────────────────

DESIGN NOTES:

Colors:
- Background: #1a1a1a (dark gray, almost black)
- Primary text: #e0e0e0 (light gray)
- Borders: #8b0000 (dark red)
- Button hover: #b22222 (firebrick)
- ASCII art: #00ff00 (terminal green)

Typography:
- Title: 32px, bold, serif
- Subtitle: 18px, italic
- Body: 16px, serif
- Buttons: 18px, sans-serif, bold

Layout:
- Max width: 800px
- Centered on page
- Padding: 40px
- Box shadow for depth

Accessibility:
- Tab navigation: Continue → New Game → Links
- Enter/Space to activate
- Screen reader: "Continue saved game from Dragon's Cave, 3 items collected"

───────────────────────────────────────────────────────────────────────────────────

EDGE CASES:

1. Save corrupted/invalid:
   - Show error: "Save data corrupted. Starting new game."
   - Hide "Continue" button

2. Very old save (30+ days):
   - Add note: "This save is 45 days old. Continue or start fresh?"

3. Anonymous user on new device:
   - No save available
   - Only "Start New Adventure" shown

4. Multiple tabs open:
   - Session conflict warning
   - "Another tab has a game in progress. Close other tabs first."
